Windows

Windows

Es el sistema operativo insignia de Microsoft mayormente utilizado para las computadoras domésticas y de trabajo, es un sistema operativo basado en una interfaz gráfica de usuario (GUI) que fue introducido en 1985 a la cual se le nombró Windows 1.0 que era esencialmente un programa que funcionaba en la parte superior de DOS.

La versión que actualmente se maneja en este sistema operativo es Windows 11, la cual el 31 de agosto de 2021 por parte de Microsoft fue anunciada que estaría disponible el 5 de octubre del mismo año.



Para comprender

¿Qué es GUI?

En inglés “Graphical user interface”, es una interfaz que se puede utilizar para controlar pc, tablets y otros dispositivos utilizan elementos gráficos como iconos, menú o imágenes con el fin de facilitar el manejo por parte del usuario. 

¿Qué es DOS?

Abreviatura de (Disk Operating System) sirvió de base para el primer sistema operativo de uso masivo, consiguieron hacer funcionar los computadores personales con una simple línea de comandos de textos.
