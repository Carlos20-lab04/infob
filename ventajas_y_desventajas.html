Ventajas y Desventajas

Ventajas

Windows - El más popular

-Tiene casi el 90% de la cuota del mercado por lo cual siempre se puede encontrar en la web soporte de utilidad para resolver un problema. 
-Casi todas las aplicaciones, controladores y juegos tienen compatibilidad con este sistema operativo. 
-Fácil instalación, administración y uso para cualquier usuario promedio. 
Interfaz intuitiva fácil de utilizar.

Macintosh - estética perfecta

-Presenta poca cantidad de virus. 
-Interfaz intuitiva, bien ordenada y con excelente estética. 
-Instalación sencilla. 
-Sistema estable, muy difícil que se corrompa. 
-Tiene su fuerte en aplicaciones para edición de videos e imágenes. 
-No tiene problemas de controladores puesto que el sistema y el hardware son incorporados por la misma compañía, por lo cual explota bien sus funcionalidades.

Linux - pequeño pero creciente 

-Se puede descargar, modificar, distribuir sin gastar un centavo 
-A pesar de ser más vulnerable a virus por ser código abierto, linux tiene poco malware. 
-Linux está basado en UNIX, por lo cual incorpora muchas de las bondades de este sistema operativo.



Desventajas

Windows

-Hay que realizar re-instalaciones frecuentes porque el sistema se corrompe con facilidad. 
-Consume muchos recursos del sistema como RAM y HDD.
-La seguridad contra malware no es muy buena, aunque está mejorando, existen casi 1 millón de virus dispuestos a atacar.
-Hay precios por cada versión del sistema operativo, por la suit de office, por antivirus, licencias entre otras lo que lo hace un sistema operativo algo costoso.

MacOS

-Es un sistema operativo muy caro. 
-Solo está disponible para ordenadores Apple. 
-Pocos programas se pueden ejecutar en MAC y no tiene disponibilidad a muchos juegos. 
-Es complicado encontrar soporte en la web por alguna falla o problema que se presente.

Linux

-Aunque las distribuciones son cada vez más fáciles de usar, la mayoría de ellos necesitan un conocimiento por encima del promedio para trabajar. 
-No hay tiendas que distribuyan GNU/Linux ya instalado, lo más probable es que se adquiera un ordenador con otro OS, y sobre ella se instale linux. 
-La instalación del sistema o agregar nuevo hardware de entrada y salida suele ser complicado para un usuario promedio, requiere de mucha lectura e investigación.
